### RDB 文件结构

```
+---------------------------------------+
|REDIS|db_version|databases|EOF|checksum|
+---------------------------------------+

展开 databases 部分
+---------------------------------------------------+
|REDIS|db_version|database 0|database 1|EOF|checksum|
+---------------------------------------------------+
```

> 熟悉 JVM 的同学可能会说，这个跟 class 文件的布局有点像啊，确实，也是差不多的
